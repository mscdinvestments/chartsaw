# 🪚 CHARTSAW
### PPT Methodology Stock Analyzer

Live market data powered by Finnhub. Deploy in under 10 minutes.

---

## 🚀 Deploy to Vercel (Recommended — Free)

### Step 1 — Create a GitHub Account
Go to **github.com** and sign up (free).

### Step 2 — Create a New Repository
1. Click the **+** icon → **New repository**
2. Name it `chartsaw`
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload the Files
1. On your new repo page, click **uploading an existing file**
2. Drag and drop ALL the files from this folder (keep the `src/` folder structure)
3. Click **Commit changes**

### Step 4 — Deploy on Vercel
1. Go to **vercel.com** → Sign up with your GitHub account
2. Click **Add New Project**
3. Select your `chartsaw` repository
4. Vercel auto-detects Vite — just click **Deploy**
5. Wait ~60 seconds

### Step 5 — Your App is Live!
Vercel gives you a URL like: `https://chartsaw-yourname.vercel.app`

**Bookmark it on your phone** — works perfectly on mobile.

---

## 📱 Add to iPhone Home Screen
1. Open the Vercel URL in **Safari**
2. Tap the **Share** button (box with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add**

CHARTSAW will appear as an app icon on your home screen.

## 📊 How to Use

1. **Enter a ticker** (e.g. HYMC, NVDA, AAPL)
2. **Click ⬇ LIVE DATA** — auto-fills price, 52W high/low, P/E, beta, EPS from Finnhub
3. **Click ▶ ANALYZE** — runs full Gareth Soloway PPT analysis
4. **Optional:** Open Advanced Parameters to set MA posture, volume trend, trendline status, and catalysts for sharper signals
